# はじめに
日本語入力の能力を最大限に引き出すためのコマンドを紹介します。 
## 矢印
zを入力してvimのキーボード操作と同じ
← `z + h`
↓ `z + j`
↑ `z + k`
→ `z + l`


## 日本語かな・カタカナ・全角英数・半角英数
キーボードで任意の文字を入力して確定していない状態で使えるコマンド
### 寿司の例
すし 日本語かな control + j
スシ カタカナ　control + k
ｓｕｓｉ 全角英数 control + l
susi 半角英数 control + ;
