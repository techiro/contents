# はじめに
Github を用いてレビューを行なってもらうことがあります。
そのレビューを行なってもらう時に、明らかにマウス操作ではできない動きでレビューをしてもらうので、
どんなコマンドがあるのだろうかと疑問に思ったため、備忘録的にコマンドを残します。


### ファイル検索
`t ＋検索したいファイル名`

### issue一覧
`g + i`

### issue 絞り込み
issue 一覧画面に遷移、＋ `a` で assignee から絞り込みができます。
`g + i + a`

### 通知一覧
`g + n`
