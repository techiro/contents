# はじめに
Githubを用いてレビューを行なってもらうことがあります。
そのレビューを行なってもらう時に、明らかにマウス操作ではできない動きでレビューをしてもらうので、
どんなコマンドがあるのだろうかと疑問に思ったため、備忘録的にコマンドを残していきたいと思います。


### ファイル検索
T ＋検索したいファイル名

### issue一覧
g + i

### issue 絞り込み
issue一覧画面に遷移、＋ `a`でassigneeから絞り込みができる
g + i + a

### 通知一覧
g + n
